# Referencias visuales

Guarda aqui capturas o fotos que sirvan como referencia para el estilo visual del juego.

Uso recomendado:

- Comparar colores.
- Comparar distribucion del tablero.
- Revisar que el jugador, enemigos, monedas y zonas se parezcan al juego original.
- Sustentar que la interfaz se inspiro en la guia del proyecto.

No es necesario cargar estas imagenes dentro del juego en la Version 1. El tablero se dibuja directamente con Java Swing en `BoardPanel`.
